INSTRUCOES PARA ENVIAR SEU PROJETO PARA O GITHUB
===========================================

1. CRIE UM REPOSITORIO NO GITHUB
   - Acesse https://github.com/new
   - Nome do repositorio: Site-de-Filmes-e-Series
   - Descricao: Plataforma de streaming para filmes e series
   - Mantenha como Publico
   - NAO inicialize com README, .gitignore ou License
   - Clique em "Create repository"

2. CRIE UM TOKEN DE ACESSO PESSOAL
   - Acesse https://github.com/settings/tokens
   - Clique em "Generate new token (classic)"
   - Note: "Token para Site de Filmes e Series"
   - Selecione os escopos: "repo" (acesso completo aos repositorios)
   - Validade: escolha conforme sua preferencia
   - Clique em "Generate token"
   - COPIE o token gerado (ele so aparece uma vez)

3. ENVIE SEU CODIGO USANDO O TOKEN (METODO 1 - CMD ou PowerShell)
   - Abra um CMD ou PowerShell e navegue ate a pasta do projeto:
     cd C:\xampp\htdocs\Player
   
   - Execute os comandos (substitua YOUR_TOKEN pelo token que voce copiou):
     git remote set-url origin https://Rakjsu:YOUR_TOKEN@github.com/Rakjsu/Site-de-Filmes-e-Series.git
     git push -u origin master

4. ENVIE SEU CODIGO (METODO 2 - GitHub Desktop)
   - Baixe e instale o GitHub Desktop: https://desktop.github.com/
   - Abra o GitHub Desktop e faca login com sua conta
   - Adicione o repositorio local (File > Add local repository)
   - Selecione a pasta C:\xampp\htdocs\Player
   - Clique no botao "Publish repository"
   - Selecione sua conta e digite "Site-de-Filmes-e-Series"
   - Clique em "Publish repository"

5. ENVIE SEU CODIGO (METODO 3 - Upload Direto)
   - Acesse seu repositorio vazio no GitHub
   - Clique em "uploading an existing file"
   - Arraste e solte os arquivos do seu projeto
   - Adicione uma mensagem de commit: "Primeiro commit: Site de Filmes e Series"
   - Clique em "Commit changes"

DEPOIS DE ENVIAR SEU CODIGO:
- Seu projeto estara disponivel em: https://github.com/Rakjsu/Site-de-Filmes-e-Series
- Voce pode compartilhar este link com outras pessoas
- Para clonar o projeto em outro computador, use: git clone https://github.com/Rakjsu/Site-de-Filmes-e-Series.git
